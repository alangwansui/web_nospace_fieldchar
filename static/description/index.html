FieldChar prohibit input space

Some very import char,filed, example: product.code, partner.cod, don't allow space in the filed, because product code 'A001' and code 'A001 ' is difficult to distinguish them. Use this module, you can defined field as:

<field name='code' class='field_char_not_space'>

When user input as 'A001 ', it will auto delete the space, trans to 'A001'.
